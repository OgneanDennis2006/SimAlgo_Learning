Bubble Sort

Teorie

Bubble Sort este un algoritm de sortare simplu care funcționează prin iterarea repetată printr-o listă, comparând elementele adiacente și schimbându-le dacă sunt în ordine greșită. Acest proces se repetă până când lista este sortată.


int n, v[100];
//citire v[] cu n elemente
bool sortat;
do
{
    sortat = true;
    for(int i = 0 ; i < n - 1 ; i ++)
        if(v[i] > v[i+1])
        {
            int aux = v[i];
            v[i] = v[i+1];
            v[i+1] = aux;
            sortat = false;
        }
}
while(!sortat);
